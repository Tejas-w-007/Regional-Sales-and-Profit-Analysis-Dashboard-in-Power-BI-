📊 Regional Sales and Profit Analysis Dashboard – Power BI
This project presents an interactive Power BI dashboard designed to visualize and analyze regional sales and profit trends across different cities and product categories.

🔍 Project Overview
Interactive Dashboard:
Built a dynamic Power BI dashboard to monitor sales and profitability by region, city, and product category.

DAX Calculations:
Used DAX (Data Analysis Expressions) to compute key metrics such as total sales, profit margins, average order value, and category-wise performance.

Insight Discovery:

Identified high-performing regions and product segments.

Pinpointed loss-making areas like Seattle vs. Houston for strategic decision-making.

🧰 Tools & Technologies
Power BI

DAX

Excel/CSV (for data source)

📌 Key Insights
Visual comparison of profit performance across regions and cities

Recognition of top-selling and underperforming product categories

Strategic insights for improving sales in low-profit zones
